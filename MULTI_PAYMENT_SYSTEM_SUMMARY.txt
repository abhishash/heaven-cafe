================================================================================
                           CHECKOUT FLOW
================================================================================

1. Customer adds items to cart
   ↓
2. Clicks "Checkout" → /checkout page
   ↓
3. Selects payment method:
   • Cash on Delivery
   • Stripe
   • Razorpay
   ↓
4. Provides necessary information:
   • COD: Delivery address, phone, email
   • Stripe: Card details in secure modal
   • Razorpay: Payment method in secure modal
   ↓
5. Confirms and processes payment
   ↓
6. Redirected to /order-confirmation
   ↓
7. Order is confirmed, cart is cleared

================================================================================
                       TROUBLESHOOTING
================================================================================

"Payment method not appearing"
  → Clear browser cache (Ctrl+Shift+Delete)
  → Restart dev server
  → Check console (F12) for errors

"Razorpay not working"
  → Verify API keys are set in environment variables
  → Restart dev server after adding keys
  → Check NEXT_PUBLIC_RAZORPAY_KEY_ID is set

"Order not confirming after payment"
  → Check browser console (F12) for errors
  → Check network tab for failed requests
  → Verify redirect URL in code

"Payment processing very slow"
  → Check internet connection
  → Check API keys are correct
  → Check payment provider status (stripe.com, razorpay.com)

For more help: See documentation files in project root
